Python CLI To-Do App
A simple, robust command-line interface for managing daily tasks. Built to practice loop logic, error handling (NaN validation), and clean functional programming.

Features:
Add Tasks: Quickly append new items to your list.

View Tasks: Check your current progress anytime.

Delete Tasks: Tasks can be deleted from the list

Input Validation: Handles non-numeric inputs gracefully.
